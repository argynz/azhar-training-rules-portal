# Corporate Training Rules Portal

MVP корпоративного портала ознакомления с Правилами обучения и развития работников, созданный на основе `PRD.md`.

## Возможности

- Главная панель с актуальной версией документа и последними изменениями.
- Раздел Правил с поиском, фильтрацией, печатью и скачиванием текстовой версии.
- Подтверждение ознакомления с фиксацией версии и даты в `localStorage`.
- История изменений документа.
- Интеллектуальный помощник, отвечающий по локальной базе пунктов Правил с указанием источника.
- HR-отчет по ознакомлению сотрудников и список уведомлений.

## Технологии

- React
- Vite
- Material UI
- lucide-react

## Запуск

```bash
npm install
npm run dev
```

## Проверка сборки

```bash
npm run build
```
